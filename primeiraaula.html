Adobe DreamWeaver

    O que é e para que serve o DreamWeaver?

    É um programa para desenvolvimento web.

    Permite a criação de sites, páginas na internet, aplicativos para dispositivos móveis e etc...

Como trabalhamos ou fazemos?

    Podemos criar os conteúdos dos sites incorporando os elementos por itens nos Menus (COMANDO), ou criamos o site programando (tags) com códigos em HTML, CSS e outras linguagens de programação web.
